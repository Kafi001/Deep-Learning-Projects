# Deep Learning Projects

This repository contains Deep Learning projects completed during my MSc in Artificial Intelligence.
The projects focus on convolutional neural networks, transfer learning, and real-time object detection.

Author: Abdullah Al Kafi  
Location: Manchester, UK

## Projects Overview

### Image Classification
- Designed and trained CNN-based image classification models.
- Applied transfer learning using MobileNetV2 and InceptionV3.
- Evaluated models using accuracy, loss curves, and confusion matrices.

### Object Detection
- Implemented real-time object detection using YOLO models.
- Focused on detection accuracy, inference speed, and deployment readiness.
- Applied data augmentation and performance evaluation (mAP, precision, recall).


## Skills Demonstrated
- Deep neural network design
- Convolutional Neural Networks (CNNs)
- Transfer learning
- Model evaluation and optimization
- GPU-based training workflows
